## 安装与运行

安装章节保留虚拟环境作为推荐方式，并提供完整 PowerShell 命令：

```powershell
python -m venv .venv
.\.venv\Scripts\python.exe -m pip install -e ".[test]"
```

同时提供使用全局 Python 的备选命令，并明确它会修改共享 Python 环境：

```powershell
python -m pip install -e ".[test]"
```

运行章节分别列出：

```powershell
# 推荐：项目虚拟环境
.\.venv\Scripts\python.exe -m flow_runner.app

# 备选：全局 Python
python -m flow_runner.app

# 安装后的命令行入口
flow-runner
```

运行说明明确默认启动加载 `data/project.json`，备份、模板、录制和日志写入 `data/` 对应子目录。
